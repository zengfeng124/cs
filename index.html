项目背景了解指南

作为助理人员，接到新项目时，请按照以下步骤快速了解项目背景：

核心原则

了解项目背景不是看完资料就行，而是要能用自己的话把项目说清楚。如果你不能简洁地概括这个项目，说明还没真正理解。

第一步：阅读项目基础资料

  • 业务约定书：了解评估目的、评估对象、评估范围
  • 委托方提供的资料清单：了解已获取和待获取的资料
  • 项目立项表：了解项目预算、时间节点、人员分工
  • 历史工作底稿：如有续作，先看过往报告和底稿

第二步：理清关键信息（用 5W1H）

项目背景五问

1. What（评什么）：评估对象是什么？资产类型？ 2. Why（为什么评）：评估目的是什么？交易？抵押？财务报告？ 3. Who（委托方是谁）：委托方背景？是否有特殊要求？ 4. When（时间节点）：基准日？报告日？何时需要提交？ 5. Where（资产在哪）：资产分布？是否需要现场勘查？ 6. How（怎么评）：评估方法？是否有特殊技术难点？

第三步：识别项目风险点

  • 资产是否存在权属瑕疵？
  • 评估目的是否涉及关联交易？
  • 是否有历史遗留问题需要关注？
  • 时间是否充裕？是否存在赶报告风险？
  • 委托方是否有特别的要求或期望？

常见问题速查

房地产评估常见问题
  • 土地性质如何确定？ → 查阅土地出让合同、房产证、不动产权证，必要时到自然资源局查询
  • 容积率怎么算？ → 建筑面积÷用地面积，以规划许可证为准
  • 收益法租金怎么取？ → 查周边同类物业实际成交租金，至少取3个可比案例
  • 抵押评估需要注意什么？ → 法定优先受偿款、查封情况、租赁情况必须核查
机器设备评估常见问题
  • 设备折旧年限怎么确定？ → 参考企业所得税法规定年限，结合设备实际使用状况
  • 进口设备汇率怎么取？ → 以基准日中国人民银行公布的外汇牌价中间价为准
  • 自制设备怎么评？ → 重置核算法：材料费+人工费+制造费用+合理利润
  • 设备已停用还能评吗？ → 可以，但需关注实体性贬值和经济性贬值
无形资产评估常见问题
  • 专利有效期怎么算？ → 发明专利20年、实用新型10年、外观设计15年，自申请日起算
  • 商标评估要注意什么？ → 是否续展、是否存在质押、许可使用情况
  • 软件著作权怎么评？ → 成本法或收益法，需了解开发成本或预期收益
企业价值评估常见问题
  • 收益法折现率怎么确定？ → WACC或CAPM模型，参数需有依据
  • 非经营性资产怎么剥离？ → 单独识别、单独评估，在结论中加减
  • 溢余资产是什么？ → 超过正常经营需要的资产，如多余现金、闲置房产
  • 控股权溢价怎么考虑？ → 需根据具体交易背景分析，一般在市场法收益法中考虑
